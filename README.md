# StepWise

Chrome extension that automates step-by-step documentation with screenshots, annotations, and permanent hosting.

## Structure

- **`/extension/`** — Chrome extension (load unpacked in chrome://extensions)
- **`/server/`** — VPS server code (publish API + deploy webhook)
- **`CLAUDE-INSTRUCTIONS.md`** — Full project context for AI assistants
- **`SETUP-GUIDE.md`** — One-time setup instructions

## Quick Start

See `SETUP-GUIDE.md` for first-time setup.

## For AI Assistants

Read `CLAUDE-INSTRUCTIONS.md` before making any changes. It has the full architecture, API docs, port map, and rules.
